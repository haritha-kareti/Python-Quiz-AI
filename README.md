<h1 align="center">🧠 AI Quiz Generator</h1>

<p align="center">
A simple Flask-based Quiz Web App with MCQ and Direct Questions
</p>

---

<h2>📌 About Project</h2>

<p>
This project is a <b>Flask Web Application</b> that allows users to attempt quizzes.
It contains both MCQ and Direct questions and shows score, percentage, and grade.
</p>

---

<h2>🚀 Features</h2>

<ul>
  <li>✔ 5 Direct Questions + 5 MCQ Questions</li>
  <li>✔ One-by-one or full quiz display</li>
  <li>✔ Automatic answer checking</li>
  <li>✔ Score calculation</li>
  <li>✔ Percentage and Grade system</li>
</ul>

---
